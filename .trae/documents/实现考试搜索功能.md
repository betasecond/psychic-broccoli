# 实现考试搜索功能

## 1. 后端修改
- **文件**: `backend/handlers/exams.go`
- **功能**: 修改 `GetExams` 函数，添加支持按考试名称搜索的功能
- **实现**: 在查询中添加 `title` 参数支持，使用 LIKE 语句进行模糊搜索

## 2. 前端修改
- **文件**: `frontend/src/services/examService.ts`
- **功能**: 修改 `getExams` 方法，添加搜索参数支持
- **实现**: 在 `params` 中添加 `title` 字段，用于传递搜索关键词

- **文件**: `frontend/src/pages/teacher/exams/ExamsListPage.tsx`
- **功能**: 添加搜索框和搜索功能
- **实现**: 
  1. 添加搜索输入框组件
  2. 添加搜索状态管理
  3. 调用 `examService.getExams` 方法，传递搜索关键词
  4. 实现搜索结果的展示

## 3. 技术栈
- 后端: Go + Gin
- 前端: React + TypeScript + Ant Design

## 4. 实现步骤
1. 首先修改后端 API，支持搜索功能
2. 然后修改前端服务层，添加搜索参数
3. 最后修改前端页面，添加搜索组件和功能

## 5. 预期效果
- 教师可以在考试列表页面输入关键词搜索考试
- 搜索结果实时展示在表格中
- 支持模糊搜索考试名称
